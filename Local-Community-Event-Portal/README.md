# Local Community Event Portal

## Project Overview

The Local Community Event Portal is a lightweight web application developed to help residents stay informed about community activities, register for events, provide feedback, and access local services. The portal demonstrates the implementation of HTML5, CSS3, JavaScript, jQuery, AJAX, Fetch API, Geolocation API, Local Storage, and Responsive Web Design concepts.

This project was developed as part of the Cognizant Digital Nurture Upskilling Program.
